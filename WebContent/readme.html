Read Me

Program do rezerwowania stolika w restauracji


Start programu

Program rozpoczyna /index.jsp

RESTful web service do wypisania listy rezerwacji rozpoczyna /api/rest


Podstawowe założenia

Rezerwacji dokonywać można w godzinach od 08:00 do 22:59 od 01/01/2018 do 31/12/2099. Rezerwacje rozpoczynają się od pełnej godziny i kończą się po jej upływie.

W restauracji dostępnych jest 5 stolików na każdą godzinę.


Walidacja danych odbywa się na poziomie przeglądarki użytkownika i jest wykonywana przez wurażenia regularne w JavaScript

Po stronie serwera nie ma walidacji wprowadzanych danych

SQL

Do obsługi bazy danych użyto MySQL

parametry połączenia to:

hostname = "localhost"

database = "restauracja"

username = "root"

rootPassword = "SQLRoot"


inicjalizacja tabeli w MySQL przy pomocy narzedzia.StartSQL


W bazie, dane przechowywane są w jednej tabeli o nazwie "rezerwacja". Kluczem głównym jest "idRezerwacja".
